# rime-files

我的 Rime 配置文件和词库。

使用的是 Windows 小狼毫输入法。

## 使用方法

下载文件，复制到用户文件夹，也可能是安装目录下的 `data` 文件夹。

## 不包含的文件

以下文件不包含在本仓库中。

- `default.yaml`
- `installation.yaml`
- `user.yaml`
- `weasel.yaml`
- `symbols.yaml`
- 各种 `userdb` 。

以下文件包含在我的另一个仓库中，见 [LS-Hower/rime-wubi86-ext](https://github.com/LS-Hower/rime-wubi86-ext)。

- `wubi86.basiccmpl.dict.yaml`
- `wubi86.extacmpl.dict.yaml`
- `wubi86.extbcmpl.dict.yaml`
- `wubi86.extc.dict.yaml`
- `wubi86.extccmpl.dict.yaml`
- `wubi86.extd.dict.yaml`
- `wubi86.exte.dict.yaml`
